# iniflex-teste-projedata
Projeto Java Maven desenvolvido para o teste prático Iniflex/Projedata. Inclui classes Pessoa e Funcionário, herança, manipulação de datas e BigDecimal, formatação pt-BR, agrupamentos, ordenação, cálculos de salários, aniversariantes, mais velho e regras de negócio simulando ERP.
